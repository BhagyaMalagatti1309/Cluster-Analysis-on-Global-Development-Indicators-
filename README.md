# Cluster-Analysis-on-Global-Development-Indicators-
This project analyses socio-economic data from countries worldwide to discover hidden patterns and groupings using unsupervised machine learning. Rather than manually categorising countries, clustering algorithms automatically group nations with similar development profiles — helping policymakers and analysts identify trends across regions.

What I did
Cleaned and normalised a multi-country socio-economic dataset (handling missing values, outliers, and feature scaling)
Applied K-Means clustering to group countries based on development indicators like GDP, literacy rate, health metrics, and income levels
Used Elbow Method and Silhouette Score to find the optimal number of clusters
Visualised cluster distributions using scatter plots, heatmaps, and pair plots
Derived actionable insights — e.g., identifying which country groups need development intervention vs which are high-performing

Tools & Technologies
Python Pandas NumPy Scikit-learn Matplotlib Seaborn
